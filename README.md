## Setup

This project require

-   [Node.js](https://nodejs.org/)
-   PostgreSQL (you can change according to your liking)

## Usage

-   Install packages

```
npm install
```

-   Change database connection in .env file file
-   Migrate prisma scheme

```
npx prisma migrate dev
```

-   Run in development

```
npm run dev
```
